参考：https://www.jianshu.com/p/01d832e028ea
第一步：先安装vue插件:
file –> settings –> plugins，然后什么都不用输入，直接点击Browse repositories… 紧接着左上方输入“vue”，点击搜索结果里的vue.js，右边会有绿色install按钮，安装成功后重启idea，这样idea就能识别.vue文件了。
