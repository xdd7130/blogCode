发布网络分析服务

步骤如下：

1.利用ArcMap创建网络（如何创建网络，请查看ArcGIS帮助文档）
2.将创建的地图进行符号化
3.开启网络分析权限,添加分析图层
4.修改属性，将其发布为网络服务
